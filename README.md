
A simple Python script to calculate permutations and combinations.

## Usage

1. Clone the repo:
    ```sh
    git clone https://github.com/r9aminu/AD-315-perm-comb.git
    cd comb_perm
    ```

2. Run the script:
    ```sh
    python comb_perm.py
    ```

3. Enter values for `n` and `r` when prompted.
